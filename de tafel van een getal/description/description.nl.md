# De tafel van een getal

## Gevraagd:

* Schrijf een programma dat de tafel van een bepaald getal toont (beginnen bij '1 maal' en eindigend bij '20 maal').

## Invoer:
```
Van welk getal wil je de tafel zien?: 5
```

## Uitvoer:

```
1.5 = 5
2.5 = 10
3.5 = 15
4.5 = 20
5.5 = 25
6.5 = 30
7.5 = 35
8.5 = 40
9.5 = 45
10.5 = 50
11.5 = 55
12.5 = 60
13.5 = 65
14.5 = 70
15.5 = 75
16.5 = 80
17.5 = 85
18.5 = 90
19.5 = 95
20.5 = 100
```
